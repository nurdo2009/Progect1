## 🎯 СРАЗУ К ДЕЛУ

### Шаг 1: Запустить проект (2 мин)
```bash
cd app
npm install  # Уже установлены firebase, react, vite
npm run dev
# Откройте http://localhost:5173
```

### Шаг 2: Настроить Firebase (5 мин)
1. Создайте проект на https://console.firebase.google.com
2. Создайте Firestore Database
3. Скопируйте `firebaseConfig`
4. Вставьте в `src/firebase/firebase.js`

### Шаг 3: Готово! 🎉
Приложение работает с real-time обновлениями
