🚀 КАК ЗАПУСТИТЬ
────────────────────────────────────────────────────────────────

1. Установить зависимости:
   npn install

2. Добавить Firebase конфиг:
   - Создать проект на https://console.firebase.google.com
   - Скопировать firebaseConfig
   - Вставить в src/firebase/firebase.js

3. Запустить dev сервер:
   npm run dev
   
4. Открыть http://localhost:5173

✅ ГОТОВО!
